# Бустер методиста PRO

Набор инструментов для L&D-методиста с AI-генерацией. Бесплатно с лимитом 3 AI-генерации/день. Безлимит через поддержку на Boosty.

## Инструменты (6)

- **Страдания инженера Иванова** — симулятор учебного дня
- **Активности** — AI-генератор активностей (3/день бесплатно)
- **Карты** — фасилитационные вопросы, 5 категорий
- **Аудит** — диагностика запроса заказчика (Mager & Pipe)
- **Видение** — мастер видения продукта по Lean/JTBD
- **AI-инструменты** — конструктор промптов

## Модель

| Тариф | Цена | AI-генерации |
|---|---|---|
| Бесплатно | 0 ₽ | 3/день |
| Сторонник | 390 ₽/мес Boosty | безлимит |

## Как разблокировать безлимит

1. Поддержать проект на [Boosty](https://boosty.to/natalialnd)
2. Получить код доступа в письме от Boosty
3. Ввести код на странице инструмента Активности

## Стек

HTML, CSS (tokens/base/components/pages), JS. GitHub Pages.
Netlify не используется — всё на статике. AI-генерация при недоступности бэкенда использует встроенную базу активностей.
