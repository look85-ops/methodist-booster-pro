AI-инструменты для методиста

Подборка AI-сервисов, которые ускоряют работу методиста.

ChatGPT (OpenAI)

Универсальный AI. Хорош для текстов, сценариев, анализа.

YandexGPT

Российский AI. Подходит для работы с русскоязычными текстами.

GigaChat (Сбер)

Российский AI. Интеграция с экосистемой Сбера.

Perplexity

Поисковый AI с источниками. Для исследований и факт-чекинга.

Claude (Anthropic)

Сильный в анализе документов и длинных текстов.

Gamma.app

AI-генерация презентаций по промпту.
